# 🌍 Disaster Resource Management System

## 📖 Overview

The **Disaster Resource Management System** is a web-based application designed to manage and allocate resources efficiently during disaster situations such as floods, earthquakes, and emergencies.

It provides a centralized platform where users can request help and administrators can monitor, verify, and distribute resources effectively.

---

## 🎯 Objective

The main objective of this project is to improve disaster response efficiency by:

* Managing resource availability
* Handling emergency requests
* Ensuring timely distribution to affected areas

---

## 🚀 Features

* 📝 User request submission system
* ✅ Admin verification of requests
* 📊 Resource allocation and tracking
* ⚡ Priority-based request handling (urgent cases first)
* 🖥️ Simple and user-friendly interface

---

## 🛠️ Tech Stack

* **Backend:** Python (Flask)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/Disaster-Resource-Management-System.git
cd Disaster-Resource-Management-System
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the application

```bash
python app.py
```

### 4️⃣ Open in browser

```
http://127.0.0.1:5000/
```
